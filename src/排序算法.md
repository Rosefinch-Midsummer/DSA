## 选择排序

- 从所有元素中找到最小的元素放到数组的第0个位置
- 从剩余元素中找出最小的放到第1个位置，以此类推，直到所有元素都放到了适当的位置

```cpp
#include <iostream>

using namespace std;

int main() {

    int lh,rh,k,tmp;
    int array[]={2,5,1,9,10,0,4,8,7,6};
    for(lh=0;lh<10;lh++)
    {
        rh=lh;
        for(k=lh;k<10;++k)
            if(array[k]<array[rh])
                rh=k;
        tmp=array[lh];
        array[lh]=array[rh];
        array[rh]=tmp;
    }
    for(lh=0;lh<10;lh++) cout<<array[lh]<<' ';

    return 0;
}
```

## 冒泡排序

- 大数沉淀、小数冒泡

```cpp
#include <iostream>

using namespace std;

int main() {

    int i,j,tmp,n=10;
    int array[]={9,8,5,4,2,0,10,6,11,7};

    for(j=0;j<n-1;j++)
        for(i=0;i<n-1-j;i++)
            if(array[i]>array[i+1])
            {
                tmp=array[i];
                array[i]=array[i+1];
                array[i+1]=tmp;
            }
    cout<<endl;

    for(i=0;i<n;i++) cout<<array[i]<<' ';

    return 0;
}
```
改进版：增加一个标志变量，若数组已排好序，则只用扫描一遍

```cpp
#include <iostream>

using namespace std;

int main() {

    int i,j,tmp,n=10;
    bool flag;
    int array[]={9,8,5,4,2,0,10,6,11,7};

    for(j=0;j<n-1;j++){
        flag=false;
        for(i=0;i<n-1-j;i++)
            if(array[i]>array[i+1])
            {
                tmp=array[i];
                array[i]=array[i+1];
                array[i+1]=tmp;
                flag=true;
            }
        if(!flag) break;
    }
    cout<<endl;

    for(i=0;i<n;i++) cout<<array[i]<<' ';

    return 0;
}
```
